# WechselWiki

## Projekt

Das WechselWiki ist ein Wissensspeicher zum Thema Wechseljahre/Menopause, der bereits verfügbare Informationen aus verschiedenen Quellen (Bücher, Zeitschriften, Webseiten, Foren, SocialMedia Posts, TV-Beiträge) sammelt, zusammenfasst, miteinander verknüpft und um eigene Beiträge ergänzt. Es stellt diese Informationen auf leicht zugängliche und recherchierbare Weise dar und ist über http://wechsel.wiki erreichbar.

## Repository

Dieses Repository dokumentiert monatlich alle im Wiki enthaltenen vollständigen Einträge als Liste in einer statischen HTML-Datei.

## Community

Das Repository dient auch als Anlaufpunkt für die WechselWiki-Community, um Fragen & Probleme, Feature Requests und Zuarbeiten zu posten und deren Bearbeitung nachzuverfolgen ([Eine Frage stellen](https://github.com/3113nbr4/wechselwiki/issues/new)). Ausserdem können Themen rund um das Wiki diskutiert werden ([Eine Diskussion starten](https://github.com/3113nbr4/WechselWiki/discussions/new/choose)).

## Aktueller Status

Über den inhaltlichen Stand des WechselWikis informiert der [WechselWiki Newsletter](https://buttondown.com/wechselwiki).


## Abhängigkeiten

Das WechselWiki basiert auf TiddlyWiki 5.
